Port Scanner

Script educativo en Python para comprobar la respuesta de puertos comunes en un host.

Ver en GitHub Descargar ZIP
Tecnologías: Python 3 • socket • uso en laboratorio
Importante: Solo usar en entornos controlados (tu equipo, VMs o laboratorios). Escanear hosts sin permiso puede ser ilegal.

Resumen

El script intenta conectar a una lista de puertos comunes (21, 22, 23, 80, 443, 3389) y reporta si responden. Ideal para aprender conceptos básicos de redes.

Cómo probar (local)

  1. Clona el repo: 
    git clone https://github.com/siralex23/portscanner.git
cd portscanner
  2. Opcional: crea y activa un entorno virtual: 
    python3 -m venv env
source env/bin/activate
  3. Levanta un servidor de prueba (puerto 8000) en otra terminal: 
    python3 -m http.server 8000
  4. Ejecuta el scanner: 
    python3 portscanner.py
# y cuando pida, escribe: localhost

Salida de ejemplo

[*] Escaneando 6 puertos comunes en localhost (127.0.0.1)...
Puerto 22: cerrado
Puerto 80: abierto
Puerto 443: cerrado
...
